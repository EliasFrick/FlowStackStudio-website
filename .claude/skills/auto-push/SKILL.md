# Auto Push to Main

## Description
After completing any code changes, automatically commit and push all changes directly to the `main` branch.

## When to Use
After any code modification, file creation, or file deletion in this project.

## Instructions
1. Stage all changes: `git add -A`
2. Commit with a concise message describing the changes
3. Push directly to `main`: `git push origin main`

Do NOT create branches or PRs. Push straight to `main`.
